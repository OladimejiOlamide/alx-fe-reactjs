GitHub User Search Application

This is a React-based application built with Vite and Tailwind CSS that allows users to search for GitHub profiles using the GitHub API.

Project Foundation

Framework: React (Vite)

HTTP Client: Axios

Icons: Lucide React

Styling: Tailwind CSS

Getting Started

Clone the repository

Navigate to github-user-search

Install dependencies: npm install

Create a .env file and add your VITE_APP_GITHUB_API_KEY

Run the dev server: npm run dev

Features

Search GitHub users by username

View profile statistics (Repos, Followers, Following)

Responsive design for mobile and desktop

Dark mode support
